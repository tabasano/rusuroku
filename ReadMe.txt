    Win32 アプリケーション: Win32Project1 プロジェクトの概要
========================================================================

この Win32Project1 アプリケーションは、AppWizard により作成されました。

このファイルには、Win32Project1 アプリケーションを構成する各ファイルの内容の概要が含まれています。


Win32Project1.vcxproj
    これは、アプリケーション ウィザードを使用して生成された VC++ プロジェクトのメイン プロジェクト ファイルです。ファイルを生成した Visual C++ のバージョンに関する情報と、アプリケーション ウィザードで選択されたプラットフォーム、構成、およびプロジェクト機能に関する情報が含まれています。

Win32Project1.vcxproj.filters
    これは、アプリケーション ウィザードで生成された VC++ プロジェクトのフィルター ファイルです。このファイルには、プロジェクト内のファイルとフィルターとの間の関連付けに関する情報が含まれています。この関連付けは、特定のノードで同様の拡張子を持つファイルのグループ化を示すために IDE で使用されます (たとえば、".cpp" ファイルは "ソース ファイル" フィルターに関連付けられています)。

Win32Project1.cpp
    これは、メインのアプリケーション ソース ファイルです。

/////////////////////////////////////////////////////////////////////////////
AppWizard によって、次のリソースが作成されました。

Win32Project1.rc
    これは、プログラムが使用するすべての Microsoft Windows リソースの一覧です。RES サブディレクトリに格納されるアイコン、ビットマップ、およびカーソルをインクルードしています。このファイルは、Microsoft Visual C++ で直接編集できます。

Resource.h
    これは、新しいリソース ID を定義する標準のヘッダー ファイルです。このファイルの読み込みおよび更新は、Microsoft Visual C++ で行います。

Win32Project1.ico
    これは、アプリケーションのアイコン (32x32) として使用されるアイコン ファイルです。このアイコンは、メイン リソース ファイル Win32Project1.rc にインクルードされます。

small.ico
    これは、アプリケーションのアイコンの小さいバージョン (16x16) を含むアイコン ファイルです。このアイコンは、メイン リソース ファイル Win32Project1.rc にインクルードされます。

fftw

fftw3 をダウンロード。使い方を調べて必要なファイルを作成、配置。
メニュー＞表示＞その他のウィンドウ＞プロパティマネージャー＞Release|Win32＞...userをダブルクリック＞ユーザーマクロ
でマクロを作成、「fftw32Path＝ダウンロードした内容にしたがったパス」など。
「ビルド環境でこのマクロを環境変数に設定する」にチェックを入れる。
Release|Win32,Release|X64などそれぞれで設定しないといけないように見られる。
あるいはメニュー＞プロジェクト＞プロパティから構成ごとにＶＣ＋＋ディレクトリの実行可能/インクルード/ライブラリディレクトリにパスを追加。
